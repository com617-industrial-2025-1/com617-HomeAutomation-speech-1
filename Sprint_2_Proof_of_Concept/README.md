# Sprint 2 — Proof of Concept (Weeks 4–7)

This folder contains the documentation produced during Sprint 2, where the team installed Home Assistant OS, integrated the Velbus hardware, and built a working voice pipeline that could control a real light via spoken command.

## Documents

1. [`01_OS_and_Platform_Setup.md`](./01_OS_and_Platform_Setup.md) — Home Assistant OS installation, network configuration via Internet Connection Sharing, onboarding, and core add-on installation
2. [`02_Velbus_Integration.md`](./02_Velbus_Integration.md) — Hardware integration, VelbusLink module programming, action mapping, and entity configuration in Home Assistant
3. [`03_Voice_Pipeline.md`](./03_Voice_Pipeline.md) — Wyoming Protocol setup, MDAR Assistant pipeline, browser microphone permissions, and the "Hello World" voice-controlled light test
