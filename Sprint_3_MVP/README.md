# Sprint 3 — Minimum Viable Product (Weeks 8–15)

This folder contains the documentation produced during Sprint 3, where the team integrated the LLM conversation agent, built the Node-RED automation flows, implemented the MQTT messaging layer, and delivered the final integrated system.

## Documents

1. [`01_AI_Integration_and_Model_Selection.md`](./01_AI_Integration_and_Model_Selection.md) — OpenRouter integration, system prompt design, comparison of Claude / GPT-4o-mini / Gemini / Ollama models, and the decision to settle on GPT-4o-mini
2. [`02_NodeRED_Automation_Implementation.md`](./02_NodeRED_Automation_Implementation.md) — SSL startup fix, Long-Lived Access Token authentication, and the five automation flows (Motion-Triggered, All Lights On / Off, Night Mode, MQTT Reporting)
3. [`03_MQTT_Messaging_Layer.md`](./03_MQTT_Messaging_Layer.md) — Mosquitto broker configuration, topic design, JSON report schema, and the publish/subscribe pipeline
4. [`04_Final_System_Architecture.md`](./04_Final_System_Architecture.md) — Complete integrated architecture, end-to-end voice and autonomous flow descriptions, testing results against requirements, consolidated problems and solutions, and future improvements
